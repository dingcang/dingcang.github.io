typescript的安装与编译

发表于 2017-04-07 更新于 2018-07-04 分类于 前端

安装typescript

  1. npm包管理器安装

$ npm install -g typescript
2. 通过与VS 2012继承的MSI

编译typescript文件

$ tsc xxx.ts //xxx为文件名

编辑tsconfig.json

tsconfig.json用来设置ts的编译选项。可以在指定目录生成一份该配置文件

$ tsc -init

编写需要的配置项

配置文件注意点

"compilerOptions"是编译选项。

  • "module"是用来指定设置编译后的js代码，使用何种模块规范。
  • "target"是编译后的js代码遵循何种规范，可以是es3/es5/es6等等。
  • "rootDir"是一个需要注意的地方，它会告诉编译器，此目录下的文件需要经过编译。
